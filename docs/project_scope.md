# Project Scope

## Goal

Build an end-to-end batch data engineering platform using modern open-source technologies.

---

## In Scope

- Daily batch ingestion
- Data validation
- Data cleansing
- Incremental processing
- Data transformation
- Star schema modeling
- Analytics-ready datasets
- Apache Spark transformations
- Apache Airflow orchestration
- Parquet storage
- SQL reporting layer
- Logging
- Configuration management
- Automated scheduling

---

## Out of Scope

- Real-time streaming
- Machine Learning
- Recommendation Engine
- Fraud Detection
- Dashboard Development
- API Development

---

## Assumptions

- Source data is delivered once daily.
- Historical files are available.
- Source schemas remain mostly stable.
- Reporting latency of one day is acceptable.

---

## Constraints

- Batch processing only
- Open-source technologies
- Local Docker-based deployments
